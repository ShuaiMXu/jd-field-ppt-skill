Mermaid / PlantUML 不作为截图嵌入，而是拆成节点、连线、标签和分组，落成可编辑矢量对象。

Drawing Path F Markdown fenced code block → diagram AST → board-native nodes / connectors
Markdown 输入Mermaid + PlantUML
```mermaid
flowchart LR
  A[需求输入] --> B[策略拆解]
  B --> C[方案生成]
  C --> D[审验收]
  D --> E[知识回流]
```

```plantuml
@startuml
用户 -> 系统: 提交需求
系统 -> AI: 生成方案
AI --> 系统: 返回候选
系统 -> 审核: 规范检查
@enduml
```
